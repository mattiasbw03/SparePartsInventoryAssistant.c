Spare Parts Inventory Assistant
Dette program fungerer som en simpel chat-assistent, der hjælper brugeren med at finde ud af, hvilke reservedele der er på lager. Lageret indeholder tre dele: hydraulic pump, PLC module og servo motor. Programmet starter med at byde brugeren velkommen og spørger: “Which part do you need?”.
Hvis brugeren skriver navnet på en del, som findes på lager, svarer programmet: “I've got {part} here for you. Bye!” og afsluttes. Hvis delen ikke findes, svarer det: “I am afraid we don't have any {part} in the inventory” og spørger igen.
Ved særlige spørgsmål som “Do you actually have any parts?” eller “Is there anything in stock at all?” viser programmet antallet af dele samt deres navne. Programmet bliver ved med at køre, indtil en gyldig reservedel findes.
